django-marionettejs

django-marionette-js is a simple Django app to conduct Web-based django-marionettejs. For each question, visitors can choose between a fixed number of answers.

Detailed documentation is in the "docs" directory.

Quick start

  1. Add "django-marionettejs" to your INSTALLED_APPS setting like this:
